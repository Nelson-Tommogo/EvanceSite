How to Install an Electric Bulb

Installing an electric bulb is a simple process that can be done in a few easy steps. Follow the instructions below to safely install a light bulb:

  1. Turn Off Power: Before you begin, make sure to turn off the power to the light fixture you are working on. This can be done by switching off the circuit breaker or removing the fuse.
  2. Remove Old Bulb: If there is an old bulb in the fixture, gently unscrew it counterclockwise to remove it from the socket.
  3. Choose the Right Bulb: Select an electric bulb that matches the wattage and type recommended for your fixture. You can usually find this information on a label near the socket.
  4. Insert New Bulb: Carefully insert the new bulb into the socket and turn it clockwise until it is snug. Be sure not to overtighten.
  5. Turn On Power: Once the new bulb is securely in place, you can turn the power back on. Test the light switch to ensure the bulb is working correctly.

Remember to always follow safety precautions when working with electricity, and if you are unsure or uncomfortable with the installation, consider seeking the help of a professional electrician.
